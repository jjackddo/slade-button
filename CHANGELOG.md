# 패치 노트

## v1.0.0 (2026-02-05)

### ✨ 주요 기능 개선
- **UI/UX 개선**:
    - 버튼 명칭 변경: '기획 하기' → '기획하기', '대화 하기' → '대화하기'
    - 각 버튼별 상세 가이드 문구 및 *이탤릭체 예시* 추가
    - '앱 사용법 및 피드백' 버튼 추가 (Confluence 가이드 페이지 연동)
- **로직 분리**:
    - **기획하기**: 페이지 본문 없이 **사용자 기획 의도**만 전송하도록 변경
    - **대화하기**: **페이지 본문 + 사용자 질문**을 결합하여 전송
- **기능 안정성**:
    - 사이드 패널 열림 로직 단순화 및 안정화 (페이지 로딩 상태와 무관하게 즉시 열림)
    - 제미나이 프롬프트 전송 후 페이지가 초기화(새로고침)되는 문제 해결

### 🐛 버그 수정
- **Confluence 테이블 마크다운 변환 오류 해결**:
    - `turndown-plugin-gfm` 적용 및 커스텀 전처리 로직 구현
    - 헤더 없는 테이블에 강제로 `<thead>` 생성하여 표 깨짐 방지
    - 셀 내부의 문단(`<p>`, `<div>`)을 제거하고 평탄화(Flattening)하여 마크다운 표 구조 유지
- **탭 이동 시 사이드 패널 닫기 관련**:
    - 닫기 로직이 패널을 영구 비활성화하는 버그 수정 (해당 기능 롤백하여 안정성 확보)
